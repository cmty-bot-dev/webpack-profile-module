## Setup
- Add `webpack-profile-module` dependency using yarn or npm to your project
- Add `webpack-profile-module` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'webpack-profile-module',

    // With options
    ['webpack-profile-module', { /* module options */ }],
 ]
}
```
